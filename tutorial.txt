1. Download a copy of Thrift, found https://thift.apache.org

2. Follow the install guides found https://thrift.apache.org/docs/install/

3. Write a thrift file, defining an interface for your service.  See bank.thrift
    for an example.

4. The Thrift compiler is used to generate your Thrift file into source code which
    is used by the different client libraries and the server you write. To generate 
    the source from a Thrift file run
    thrift --gen <language> <Thrift filename>
